# Simple JSF Demo
## Requirements
* Java EE 8 application server
    * JSF 2.3
    * EL 3.0
* Maven 3

## Properties
* Packaging: WAR-File

## Build
* ```mvn clean package```

## Deployment
* ```target/jsfsimple.war```
* by hand (see application server manual)

## Use
* ```127.0.0.1:8080/jsfsimple/something.xhtml```

## Tested on
* Oracle Glassfish 5
